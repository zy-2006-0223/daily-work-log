# 📋 我的工作日志 - 个性化每日工作记录平台

一个功能完整的**每日工作记录平台**，集计划、待办、复盘、习惯追踪、统计于一体。

## ✨ 功能特色

### 📝 今日计划
- 设定每日「最重要的三件事」
- 添加今日任务，支持优先级和分类

### ✅ 待办清单
- 全量任务管理，按日期、状态筛选
- 支持优先级（紧急/重要/普通/低优）
- 支持分类（工作/个人/学习/健康/其他）
- 预计时间记录

### 🔄 每日复盘
- 心情评分（😢😕😐😊🥰）
- 今日完成记录
- 今日亮点
- 遇到的困难
- 改进方向
- 今日感恩（三件小事）
- 自由笔记
- 明日计划概要

### 📊 周复盘
- 本周数据概览（总任务/完成率/习惯打卡）
- 每日完成情况柱状图
- 本周成就、挑战、下周目标

### 🔥 习惯追踪
- 自定义习惯（支持图标选择）
- 每日打卡
- 连续天数统计

### 📈 统计看板
- 自定义时间范围（7/14/30/90天）
- 任务完成率趋势图
- 完成情况分布环形图
- 最长习惯打卡记录

### 📚 历史记录
- 按日期浏览所有记录
- 展开查看任务和复盘详情

### 🔄 数据导入导出
- 导出为 JSON 文件备份
- 导入恢复数据

## 🚀 如何使用

### 方法一：直接打开（最简单）
双击 `index.html` 在浏览器中打开即可使用。

### 方法二：安装为 PWA（推荐）
支持 Chrome/Edge/Safari 安装到桌面或手机主屏幕：

**电脑端：**
1. 用 Chrome 打开 `index.html`
2. 地址栏右侧点击「安装」图标（或菜单 → 安装 工作日志）

**手机端：**
1. 用 Safari/Chrome 打开 `index.html`
2. Safari：分享按钮 → 添加到主屏幕
3. Chrome：菜单 → 添加到主屏幕

安装后可以离线使用，数据存储在本地浏览器中。

## 📱 多端适配

- **桌面端**：左侧导航栏 + 内容区
- **移动端**：底部导航 + 侧边栏菜单
- 响应式设计，自动适配屏幕尺寸

## 💾 数据存储

所有数据存储在浏览器的 **LocalStorage** 中：
- 数据不会上传到任何服务器
- 完全离线可用
- 隐私安全，仅保存在本地

> ⚠️ 建议定期导出数据备份（统计页面 → 导出数据）

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+1` | 首页 |
| `Ctrl+2` | 今日计划 |
| `Ctrl+3` | 待办清单 |
| `Ctrl+4` | 每日复盘 |
| `Ctrl+5` | 周复盘 |

| `Ctrl+7` | 统计 |

## 🛠️ 技术栈

- 纯 HTML + CSS + JavaScript（无任何依赖）
- PWA（Service Worker + Manifest）
- LocalStorage 数据持久化
- 响应式设计（CSS Grid/Flexbox）
- 支持深色模式（跟随系统）
